# Banner

Banners are generic containers for displaying important information to the user. They usually contain a heading, descriptive text and a call to action.

```jsx react-live
<Banner maxWidth="400px">
  <Heading size="bravo" as="h2">
    <IconLargeMessage
      circleColor={colors.brand}
      style={{ fontSize: '3rem', marginRight: '1rem' }}
    />
    Need help?
  </Heading>
  <p>
    If you require any assistance our 24/7 live chat support is here to help
  </p>
  <Button fullWidth modifier="secondary">
    Contact Us
  </Button>
</Banner>
```

## Horizontal

Banner content can also be switched to a `horizontal` orientation.

```jsx react-live
<Banner maxWidth="800px" horizontal>
  Don't have any bitcoins?
  <Button fullWidth modifier="secondary">
    Buy bitcoins
  </Button>
</Banner>
```

## Closeable

`BannerToggle` (an extension of `Toggle`) can be used to offer users with the option to close a Banner.

```jsx react-live
<Banner maxWidth="800px" horizontal>
  Don't have any bitcoins?
  <Button fullWidth modifier="secondary">
    Buy bitcoins
  </Button>
  <BannerToggle
    onClick={() => {
      console.log('Closed');
    }}
  />
</Banner>
```

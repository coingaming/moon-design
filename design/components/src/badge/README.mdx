# Badge

Small count and labeling component

```jsx react-live

```

## Customizing colors

You coud simply use `color` and `backgroundColor` props

```jsx react-live
<Badge color={colors.error} backgroundColor={colors.neutral[10]}>
  Active
</Badge>
```

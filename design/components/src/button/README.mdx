# Button

## Modifiers

### Primary

To be used where the the button would be the main call to action on the page.

```jsx react-live
<Button modifier="primary">
  Default
</Button>
<Button modifier="primary" mockState="active" >
  Hover
</Button>
<Button modifier="primary" mockState="focus" >
  Focus
</Button>
<Button modifier="primary" disabled >
  Disabled
</Button>
```

### Secondary

To be used where the the button would be alternative or secondary call to action.

```jsx react-live
<Button modifier="secondary" >
  Default
</Button>
<Button modifier="secondary" mockState="active" >
  Hover
</Button>
<Button modifier="secondary" mockState="focus" >
  Focus
</Button>
<Button modifier="secondary" disabled >
  Disabled
</Button>
```

### Optional

To be used where the the button would be an optional call to action - meaning not prominant.

```jsx react-live
<Button modifier="optional" >
  Default
</Button>
<Button modifier="optional" mockState="active" >
  Hover
</Button>
<Button modifier="optional" mockState="focus" >
  Focus
</Button>
<Button modifier="optional" disabled >
  Disabled
</Button>
```

### Alternate

Alternate shade for buttons, currently only intended for use with "Price Boost".

```jsx react-live
<Button modifier="alternate" >
  Default
</Button>
<Button modifier="alternate" mockState="active" >
  Hover
</Button>
<Button modifier="alternate" mockState="focus" >
  Focus
</Button>
<Button modifier="alternate" disabled >
  Disabled
</Button>
```

## Sizing

### Full Width

Stretches a button to fill the entire width of its container.

```jsx react-live
<Button modifier="secondary" fullWidth>
  Full Width
</Button>
```

## Elements

### Anchor

Sometimes we want to use the Button component to send the user to a different location.

By adding the `href="<url>"` attribute, Buttons can be rendered as `<a>` elements.

```jsx react-live
<Button href="#" modifier="secondary">
  Secondary (Anchor)
</Button>
```

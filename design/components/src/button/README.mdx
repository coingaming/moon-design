# Button

## Modifiers

### Primary

To be used where the the button would be the main call to action on the page.

```jsx react-live
<Button modifier="primary">
  Default
</Button>
<Button modifier="primary" mockState="hover" >
  Hover
</Button>
<Button modifier="primary" mockState="active" >
  Focus
</Button>
<Button modifier="primary" disabled >
  Disabled
</Button>
```

### Highlight

To be used where the the button would be a highlight call to action.

```jsx react-live
<Button modifier="highlight" >
  Default
</Button>
<Button modifier="highlight" mockState="hover" >
  Hover
</Button>
<Button modifier="highlight" mockState="active" >
  Focus
</Button>
<Button modifier="highlight" disabled >
  Disabled
</Button>
```

### Secondary

To be used where the the button would be a secondary call to action.

```jsx react-live
<Button modifier="secondary" >
  Default
</Button>
<Button modifier="secondary" mockState="hover" >
  Hover
</Button>
<Button modifier="secondary" mockState="active" >
  Focus
</Button>
<Button modifier="secondary" disabled >
  Disabled
</Button>
```

## Uppercase

To be used where the the button caption would be uppercased.

```jsx react-live
<Button modifier="primary" uppercase>
  Uppercased primary
</Button>
<Button modifier="secondary" uppercase >
  Uppercased secondary
</Button>
```

## Sizing

### Full Width

Stretches a button to fill the entire width of its container.

```jsx react-live
<Button modifier="secondary" fullWidth>
  Full Width
</Button>
```

## Elements

### Anchor

Sometimes we want to use the Button component to send the user to a different location.

By adding the `href="<url>"` attribute, Buttons can be rendered as `<a>` elements.

```jsx react-live
<Button href="#" modifier="secondary">
  Secondary (Anchor)
</Button>
```

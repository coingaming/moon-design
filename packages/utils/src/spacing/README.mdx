# Spacing

Returns a given [spacing token "key"](/tokens#spacing) value as a `rem` spacing unit.

Using `spacing()` is encouraged over using the tokens directly.

```js
import { spacing } from '@heathmont/sportsbet-utils';
```

## Example

<Playground>
  {() => {
    return (
      <ul>
        <li>
          <code>spacing()</code> returns <code>{spacing()}</code>
        </li>
        <li>
          <code>spacing("small")</code> returns <code>{spacing('small')}</code>
        </li>
      </ul>
    );
  }}
</Playground>

## Usage

In a component, this could be used to create brand-conforming spacing:

```jsx
import styled from '@emotion/styled';
import { spacing } from '@heathmont/sportsbet-utils';

const Box = styled('div')`
  margin-bottom: ${spacing()};
`;
```

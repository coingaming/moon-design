# Breakout

```js
import { breakout } from '@heathmont/moon-utils';
```

Quickly jump out of any parent's constraining `maxWidth` by applying the `breakout` mixin.

```jsx react-live
() => {
  const { color, colorNew } = useTheme();

  const Example = styled.div(({ css }) => css);

  return (
    <Example css={{ ...container(), background: colorNew.piccolo }}>
      <p>Bound by the Sportsbet container</p>
      <Example css={{ ...breakout, background: color.chiChi[100] }}>
        <p>Breakout from the parent container</p>
      </Example>
    </Example>
  );
};
```

# Helpers

## Unique ID

```js
import { uniqueId } from '@heathmont/sportsbet-utils';
```

Generates a unique identifier to facilitate conformity with web content
accessibility guidelines.

Specifially useful for automatically adding `id`s to more complex areas of UI
such as forms.

<Playground>
  <p>
    <code>`Example-${'{uniqueId()}'}`</code> generates {`Example-${uniqueId()}`}
  </p>
</Playground>

## Spacing

⚠️ **Deprecated**: please use `rem` with space values from your theme.

```js
import { spacing } from '@heathmont/sportsbet-utils';
```

Returns a given [spacing token "key"](/themes/tokens#space) value as a `rem` spacing unit.

Using `spacing()` is encouraged over using the tokens directly and can be reset
as `spacing(0)`.

```jsx react-live
() => {
  const Example = styled.div(({ css }) => css);

  return (
    <>
      <Example css={{ padding: spacing(), border: '1px solid white' }}>
        Default
      </Example>
      <Example css={{ padding: spacing('small'), border: '1px solid white' }}>
        Small
      </Example>
      <Example css={{ padding: spacing('large'), border: '1px solid white' }}>
        Large
      </Example>
    </>
  );
};
```

## EM

```js
import { em } from '@heathmont/sportsbet-utils';
```

An extension of [polished's `em` function](https://polished.js.org/docs/#em),
returning a given string or number value as a `em` unit.

For example `em(16)` or `rem('16px')` will return the following:

<Playground>
  <p>{em(16)}</p>
</Playground>

## REM

```js
import { rem } from '@heathmont/sportsbet-utils';
```

An extension of [polished's `rem` function](https://polished.js.org/docs/#rem),
returning a given string or number value as a `rem` unit based on **our
[`base.fontSize` token](/themes/tokens#base)**.

For example `rem(14)` or `rem('14px')` will return the following:

<Playground>
  <p>{rem(14)}</p>
</Playground>

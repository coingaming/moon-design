# Mixins

## Breakout

```js
import { breakout } from '@heathmont/sportsbet-utils';
```

Quickly jump out of any parent's constraining `maxWidth` by applying the `breakout` mixin.

```jsx react-live
() => {
  const Example = styled.div(({ css }) => css);

  return (
    <Example css={{ ...container(), background: colors.brand }}>
      <p>Bound by the Sportsbet container</p>
      <Example css={{ ...breakout, background: colors.error }}>
        <p>Breakout from the parent container</p>
      </Example>
    </Example>
  );
};
```

## Container

```js
import { container } from '@heathmont/sportsbet-utils';
```

Returns a given [container "key"](/tokens#container) value as a `rem`-based `max-width` unit with horizontal spacing.

`container()` will also accept any pixel value (as a `number`) and convert to `rem`.

```jsx react-live
() => {
  const Example = styled.div(({ css }) => css);

  return (
    <>
      <Example css={{ ...container(), border: '1px solid white' }}>
        "default" Container
      </Example>
      <Example css={{ ...container('large'), border: '1px solid white' }}>
        "large" Container
      </Example>
      <Example css={{ ...container(300), border: '1px solid white' }}>
        300px Container (in REM)
      </Example>
    </>
  );
};
```

### Padding

By default, containers render padding on the `x` axis with the `default` spacing
token.

This padding can be altered in the second argument, either as a `spacing` token
key or a pixel value (as a `number`) which will be convert to `rem`

```jsx react-live
() => {
  const Example = styled.div(({ css }) => css);

  return (
    <>
      <Example
        css={{ ...container('default', 'large'), border: '1px solid white' }}
      >
        "default" Container and "large" padding
      </Example>
      <Example css={{ ...container('default', 50), border: '1px solid white' }}>
        "default" Container and 50px padding (in REM)
      </Example>
      <Example css={{ ...container('default', 0), border: '1px solid white' }}>
        "default" Container with no padding
      </Example>
    </>
  );
};
```

## Disabled

```js
import { disabled } from '@heathmont/sportsbet-utils';
```

Apply common disabled styles to UI elements such as buttons and forms.

`disabled()` will also accept any CSS (as an object) and return it inside disabled selectors.

```jsx react-live
() => {
  const ButtonExample = styled.button(({ css }) => css);

  return (
    <>
      <ButtonExample disabled css={{ ...disabled() }}>
        Submit
      </ButtonExample>
      <ButtonExample disabled css={{ ...disabled({ color: 'white' }) }}>
        Submit
      </ButtonExample>
    </>
  );
};
```

## Focus

```js
import { focus } from '@heathmont/sportsbet-utils';
```

Override browser-default `:focus` styles to the preferred appearance.

By default `focus()` this renders with the `brand` color, but an alternative color can be supplied.

```jsx react-live
() => {
  const ButtonExample = styled.button(({ css }) => css);

  return (
    <>
      <ButtonExample css={{ ...focus() }}>Focus on to me</ButtonExample>
      <ButtonExample css={{ ...focus('rebeccapurple') }}>
        Focus on to me
      </ButtonExample>
    </>
  );
};
```

If you just want to remove the browser default styles and create your own, you
can make use of `focusReset`. It is **strongly** advised not to use this without
alternative `:focus` styles.

```jsx react-live
() => {
  const ButtonExample = styled.button(({ css }) => css);

  return <ButtonExample css={{ ...focusReset }}>No focus styles</ButtonExample>;
};
```

## MQ

Returns a given pixel-based value (number) as an **upwards** direction media query in 'em'.

```js
import { mq } from '@heathmont/sportsbet-utils';
```

### Example

`mq(500)` will return the following:

<Playground>
  <p>{mq(500)}</p>
</Playground>

Passing the additional `mq(500, 'max-width')` parameter will return a breakpoint in the opposite direction.

<Playground>
  <p>{mq(500, 'max-width')}</p>
</Playground>

### Usage

With our breakpoint tokens, we can start to build-up our responsive styles:

```jsx
import styled from 'styled-components';
import { breakpoints } from '@heathmont/sportsbet-tokens';
import { mq } from '@heathmont/sportsbet-utils';

/* Changes border from `red` to `blue` from 'small' breakpoint upwards */

const Box = styled('div')`
  border: 1px solid red;

  ${mq(breakpoints.small)} {
    border-color: blue;
  }
`;
```

# REM

An extension of [polished's `rem` function](https://polished.js.org/docs/#rem),
returning a given string or number value as a `rem` unit based on **our
[`base.fontSize` token](/themes/tokens#base)**.

For example `rem(24)` will return the following:

```jsx react-live
<p>{rem(24)}</p>
```

## All possible sizes for _font-size_ property:

`rem(10)`, `rem(12)`, `rem(14)`, `rem(16)`, `rem(18)`, `rem(20)`, `rem(24)`, `rem(32)`, `rem(48)`, `rem(56)`, `rem(64)`, `rem(72)`.

## All possible sizes for _line-height_ property:

`rem(16)`, `rem(20)`, `rem(24)`, `rem(28)`, `rem(32)`, `rem(40)`, `rem(56)`, `rem(64)`, `rem(72)`, `rem(76)`.

## All other gaps, sizes and spaces:

`rem(4)`, `rem(8)`, `rem(12)`, `rem(16)`, `rem(20)` and other numbers which are divisible by 4 only.

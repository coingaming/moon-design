# Pagination

Pagination component based on [react-paginate](https://github.com/AdeleD/react-paginate).

[Figma link](https://www.figma.com/file/rQDkupIZuIlueXNMoYObjt/Hub88---Templates?node-id=165%3A14678)

### Props

| Name                  | &nbsp;&nbsp;&nbsp;&nbsp; |        Type         | &nbsp;&nbsp;&nbsp;&nbsp; | Description                                                                                |
| :-------------------- | ------------------------ | :-----------------: | ------------------------ | :----------------------------------------------------------------------------------------- |
| pageCount             |                          |       number        |                          | **Required**. The total number of pages.                                                   |
| pageRangeDisplayed    |                          |       number        |                          | **Required**. The range of pages displayed.                                                |
| marginPagesDisplayed  |                          |       number        |                          | **Required**. The number of pages to display for margins.                                  |
| previousButtonLabel   |                          | string, JSX.Element |                          | Label for "previous" button                                                                |
| nextButtonLabel       |                          | string, JSX.Element |                          | Label for "next" button                                                                    |
| onChange              |                          |      function       |                          | The method to call when a page is clicked. Exposes the current page object as an argument. |
| pageNumber            |                          |       number        |                          | To override selected page with parent prop                                                 |
| pageSizeSection       |                          |     JSX.Element     |                          | Placeholder for "page size" component                                                      |
| goToPageSection       |                          |     JSX.Element     |                          | Placeholder for "go to page" component                                                     |
| changePageSizeSection |                          |     JSX.Element     |                          | Placeholder for "change page size" component                                               |

## Examples

```jsx react-live
<Pagination
  pageCount={20}
  pageRangeDisplayed={3}
  marginPagesDisplayed={1}
  pageNumber={10}
  previousButtonLabel="Previous"
  nextButtonLabel="Next"
  onChange={value => console.log('pageNumber: ', value)}
/>
```

## With the "page size" section

```jsx react-live
<Pagination
  pageCount={20}
  pageRangeDisplayed={3}
  marginPagesDisplayed={1}
  pageNumber={10}
  previousButtonLabel="Previous"
  nextButtonLabel="Next"
  onChange={value => console.log('pageNumber: ', value)}
  pageSizeSection={
    <Text size={12} as="strong">
      Showing 1 – 10{' '}
      <Text color="trunks.100" size={12} as="span">
        of 20
      </Text>
    </Text>
  }
/>
```

## With the "go to page" section

```jsx react-live
<Pagination
  pageCount={20}
  pageRangeDisplayed={3}
  marginPagesDisplayed={1}
  pageNumber={10}
  previousButtonLabel="Previous"
  nextButtonLabel="Next"
  onChange={value => console.log('pageNumber: ', value)}
  goToPageSection={
    <GoToPage
      minPage={1}
      maxPage={20}
      onChange={value => console.log('GoToPage: ', value)}
    >
      Go to page
    </GoToPage>
  }
/>
```

## With the "change page size" section

```jsx react-live
<Pagination
  pageCount={20}
  pageRangeDisplayed={3}
  marginPagesDisplayed={1}
  pageNumber={10}
  previousButtonLabel="Previous"
  nextButtonLabel="Next"
  onChange={value => console.log('pageNumber: ', value)}
  changePageSizeSection={
    <ChangePageSize
      pageSize={20}
      onChange={value => console.log('Change page size: ', value)}
      options={[
        { label: '10', value: 10 },
        { label: '20', value: 20 },
        { label: '50', value: 50 },
        { label: '100', value: 100 },
      ]}
    >
      Rows: {20}
    </ChangePageSize>
  }
/>
```

# Pagination

Pagination component based on [react-paginate](https://github.com/AdeleD/react-paginate).
**_Page index is zero-based_**

[Figma link](https://www.figma.com/file/rQDkupIZuIlueXNMoYObjt/Hub88---Templates?node-id=165%3A14678)

### Props

| Name                   | &nbsp;&nbsp;&nbsp;&nbsp; |        Type         | &nbsp;&nbsp;&nbsp;&nbsp; | Description                                                                                |
| :--------------------- | ------------------------ | :-----------------: | ------------------------ | :----------------------------------------------------------------------------------------- |
| pageCount              |                          |       number        |                          | **Required**. The total number of pages.                                                   |
| pageRangeDisplayed     |                          |       number        |                          | **Required**. The range of pages displayed.                                                |
| marginPagesDisplayed   |                          |       number        |                          | **Required**. The number of pages to display for margins.                                  |
| previousButtonLabel    |                          | string, JSX.Element |                          | Label for "previous" button                                                                |
| nextButtonLabel        |                          | string, JSX.Element |                          | Label for "next" button                                                                    |
| onPageChange           |                          |      function       |                          | The method to call when a page is clicked. Exposes the current page object as an argument. |
| initialPage            |                          |       number        |                          | The initial page selected                                                                  |
| disableInitialCallback |                          |       boolean       |                          | Disable onPageChange callback with initial page. **Default: false**                        |
| itemsPerPageLabel      |                          | string, JSX.Element |                          | Component for showing items per page section. Hidden if empty                              |
| goToPageLabel          |                          | string, JSX.Element |                          | Label for "Go to page" section. Hidden if empty                                            |

## Examples

```jsx react-live
<Pagination
  pageCount={20}
  pageRangeDisplayed={3}
  marginPagesDisplayed={1}
  previousButtonLabel="Previous"
  nextButtonLabel="Next"
  onPageChange={({ selected }) => console.log(selected)}
  initialPage={10}
  disableInitialCallback={true}
/>
```

### With "items per page" section

```jsx react-live
<Pagination
  pageCount={20}
  pageRangeDisplayed={3}
  marginPagesDisplayed={1}
  previousButtonLabel="Previous"
  nextButtonLabel="Next"
  initialPage={10}
  disableInitialCallback={true}
  itemsPerPageLabel={
    <>
      Showing 1 – 10 <span>of 100</span>
    </>
  }
/>
```

### With "go to page" section

```jsx react-live
<Pagination
  pageCount={20}
  pageRangeDisplayed={3}
  marginPagesDisplayed={1}
  previousButtonLabel="Previous"
  nextButtonLabel="Next"
  initialPage={10}
  disableInitialCallback={true}
  goToPageLabel={'Go to page'}
/>
```

### With "items per page" and "go to page" sections

<Pagination

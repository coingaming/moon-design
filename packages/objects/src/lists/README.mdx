# Lists

Sometimes it makes semantic sense to build a list of items (e.g. navigation or a
form) without the traditional 'bullets'.

Removing the `list-style-type` on its own has its [caveats](https://unfetteredthoughts.net/2017/09/26/voiceover-and-list-style-type-none/).
Using these list objects will ensure all users benefit from the semantics whilst
also offering a range of layout options.

Each of these list styles can be extended in style definitions or used on the
`css` prop.

## Plain

A basic plain list with no bullets.

```jsx react-live
() => {
  const ExampleList = styled.ul(listPlain);
  const ExampleListItem = styled.li(listPlainItem);

  return (
    <ExampleList>
      <ExampleListItem>Item 1</ExampleListItem>
      <ExampleListItem>Item 2</ExampleListItem>
      <ExampleListItem>Item 3</ExampleListItem>
    </ExampleList>
  );
};
```

## Inline

Displays the list and its items inline with no bullets.

```jsx react-live
() => {
  const ExampleList = styled.ul(listInline);
  const ExampleListItem = styled.li(listInlineItem, {
    '& + &': { marginLeft: '1rem' },
  });

  return (
    <ExampleList>
      <ExampleListItem>Item 1</ExampleListItem>
      <ExampleListItem>Item 2</ExampleListItem>
      <ExampleListItem>Item 3</ExampleListItem>
    </ExampleList>
  );
};
```

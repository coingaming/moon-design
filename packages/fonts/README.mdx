# Fonts

A collection of private fonts for use within the Sportsbet design system,
currently including:

1. `avertaStd` - Averta Standard ([sportsbet.io](https://sportsbet.io/))

## Usage

1. Import the `fontFace` into your global stylesheet.

```jsx
/* e.g. Emotion */
import { avertaStd } from '@heathmont/sportsbet-fonts';
import { Global } from '@emotion/core';

export const App = () => (
  <main>
    <Global styles={avertaStd.fontFace} />
    {/* â€¦app */}
  </main>
);
```

2. Import crucial font settings into your design tokens for further
   implementation:

```js
import { avertaStd } from '@heathmont/sportsbet-fonts';

export const typography = {
  fontFamily: avertaStd.fontStack,
  fontWeight: avertaStd.fontWeight,
};
```

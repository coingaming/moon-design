# Sidebar

Collapsible sidebar component

```jsx react-live
<Sidebar
  collapseIcon={<IconChevronLeftRounded />}
  expandIcon={<IconChevronRightRounded />}
  logo={
    <Inline fontSize="2rem">
      <IconBitcoinTalk />
    </Inline>
  }
>
  Something
</Sidebar>
```

## Props

| Name          | Type                     | Description              |
| :------------ | :----------------------- | :----------------------- |
| isOpen        | boolean                  | Initial sidebar state    |
| collapseIcon  | React.ReactNode          | close sidebar icon       |
| expandIcon    | React.ReactNode          | open sidebar icon        |
| logo          | React.ReactNode          | Logo icon                |
| openWidth     | number                   | open width in pixels     |
| collapseWidth | number                   | closed width in pixels   |
| onStateChange | (state: boolean) => void | callback on state change |

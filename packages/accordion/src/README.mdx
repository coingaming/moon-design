# Accordion

| Name          | &nbsp;&nbsp; |      Type       | &nbsp;&nbsp; | Description                                                 |
| :------------ | ------------ | :-------------: | ------------ | :---------------------------------------------------------- |
| title         |              |     string      |              | **Required**. Title of accordion                            |
| openByDefault |              |     boolean     |              | **Default: false**. Expanded/collapsed accordion by default |
| withArrow     |              |     boolean     |              | **Default: true**. Show expand/collapse chevron             |
| disableOpen   |              |     boolean     |              | **Default: false**. Disable expand/collapse accordion       |
| headerContent |              | React.ReactNode |              | Additional content to render in accordion header            |
| children      |              | React.ReactNode |              | Content to show inside of accordion                         |

```jsx react-live
<Inline>
  <Accordion
    title="Test"
    headerContent={
      <div onClick={() => console.log('Clicked')}>Additional content</div>
    }
  >
    <div>
```

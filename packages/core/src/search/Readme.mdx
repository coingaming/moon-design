# Search

An elegant search component.

## Query

Use `query` prop to define query string

`onChange` and `onSubmit` handlers are provided.

<Search
  placeholder="Search"
  onChange={() => {}}
  onSubmit={() => {}}
  onClear={() => {}}
/>

### Results

Pass searchResults to `searchResults` prop.

```jsx react-live
<Search
  placeholder="Search"
  onChange={(e) => {}}
  onSubmit={(e) => {}}
  tabs={
    {
      title: <span>Categories</span>,
      items: [
        <span
          onClick={() => {}}
        >
          First cat
        </span>,
      ],
    },
  }
  results={
    {
      title: <span>Games</span>,
      items: [
        <span
          onClick={() => {}}
        >
          First Game
        </span>,
      ],
    },
  }
/>
```

### Loading Message

Pass `loadingMessage`.
loadingMessage has a priority and even if you define searchResults
they won't be rendered.

```jsx react-live
<Search
  placeholder="Search"
  onChange={(e) => {}}
  loadingMessage={<p>Loading...</p>}
/>
```

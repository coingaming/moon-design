# Age Limit

Tiny age limit labeling component.

## Usage:

```
import { AgeLimit } from '@heathmont/moon-core';
```

```jsx react-live
<Inline>
  <AgeLimit caption="18+" />
  <AgeLimit caption="21+" />
</Inline>
```

## Props

### Customize colours

You could use `color` prop.

Default colour is:

```
color="bulma.100"
```

```jsx react-live
<Inline>
  <AgeLimit caption="18+" />
  <AgeLimit caption="18+" color="bulma.100" />
  <AgeLimit caption="18+" color="piccolo.100" />
  <AgeLimit caption="18+" color="chiChi.100" />
</Inline>
```

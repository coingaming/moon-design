# Icons

Icons offer visual aids for common actions and tasks.

To aid with performance, Icons are surfaced as individual [SVG](https://developer.mozilla.org/en-US/docs/Web/SVG) components. Each icon component is prefixed with `Icon`.

### Brand

```jsx react-live
<Inline fontSize="2rem">
  <IconLogoPlain />
  <IconCasino />
  <IconLiveCasino />
  <IconAsianView />
  <IconSbShield />
  <IconBombayClub />
  <IconSlots />
</Inline>
```

## Customizing

### Color

Icon colors can be modified by adding the `color` attribute/CSS property to the
component; these can be regular CSS color values, or themed colors in dot
notation.

```jsx react-live
<Inline fontSize="2rem">
  <IconSettings fontSize="2rem" color="piccolo.100" />
  <IconSettings fontSize="2rem" color="#FFFF00" />
</Inline>
```

There's also the option to define a `backgroundColor`.

Note: for inline SVGs you may need to apply an additional `background-image` of
a circle behind the icon.

```jsx react-live
<IconClose fontSize="2rem" color="goten.100" backgroundColor="chiChi.100" />
```

### Size

Icon size can be modified by adding the `fontSize` attribute or `font-size` CSS property to the component or its parent element.

```jsx react-live
<IconSettings fontSize="3rem" />
```

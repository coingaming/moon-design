# Assets

Styled SVG components, sizeable via `fontSize` or traditional
`width` and `height` methods.

## Crests

```jsx react-live
() => {
  const Example = styled.div({
    display: 'grid',
    gridTemplateColumns: 'repeat(auto-fill, minmax(8rem, 1fr));',
    gridGap: '1rem',
  });

  return (
    <Example>
      <CrestFlamengo fontSize="10rem" />
      <CrestWatford fontSize="10rem" />
    </Example>
  );
};
```

## Logos

### Sportsbet.io

With `fontSize`:

```jsx react-live
<LogoSportsbet fontSize="10rem" />
```

With `width` and `height`:

```jsx react-live
<LogoSportsbet width="100%" height="auto" />
```

With the current sponsors:

```jsx react-live
<LogoSportsbetSponsor fontSize="16rem" />
```

## Patterns

```jsx react-live
() => {
  const Example = styled.div({
    display: 'grid',
    gridTemplateColumns: 'repeat(auto-fill, minmax(6.5rem, 1fr));',
    gridGap: '1rem',
  });

  return (
    <Example>
      <PatternCircles fontSize="5rem" />
      <PatternRipple fontSize="5rem" />
      <PatternSpeed fontSize="5rem" />
      <PatternStars width="auto" height="5rem" />
      <PatternWave width="auto" height="5rem" />
    </Example>
  );
};
```

# CheckboxMultiselect

Group of checkbox options

```jsx react-live
<CheckboxMultiselect
  onChange={value => console.log(value)}
  clearSelectedLabel={<span>Clear selected items</span>}
  value={[]}
  options={[
    { label: 'Game 1', value: 'Game 1' },
    { label: 'Game 2', value: 'Game 2' },
    {
      label: (
        <span>
          Game <b>3</b>
        </span>
      ),
      value: 'Game 3',
    },
    {
      label: (
        <span>
          Game <b>4</b>
        </span>
      ),
      value: 'Game 4',
    },
    {
      label: (
        <span>
          Game <b>5</b>
        </span>
      ),
      value: 'Game 5',
    },
  ]}
/>
```

## With inner options

```jsx react-live

  onChange={value => console.log(value)}
  clearSelectedLabel={<span>Clear selected items</span>}
  value={[]}
  options={[
    {
      label: 'Game 1',
      value: 'Game 1',
      innerOptions: [
        {
          label: 'Game 1-1',
          value: 'Game 1-1',
        },
        {
          label: 'Game 1-2',
          value: 'Game 1-2',
        },
```

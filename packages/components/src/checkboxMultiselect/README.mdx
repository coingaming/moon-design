# CheckboxMultiselect

Group of checkbox options

```jsx react-live
<CheckboxMultiselect
  onChange={(value) => console.log('value: ', value)}
  value={['Game 1', 'Game 3']}
  options={[
    { label: 'Game 1', value: 'Game 1' },
    { label: 'Game 2', value: 'Game 2' },
    { label: 'Game 3', value: 'Game 3' },
    { label: 'Game 4', value: 'Game 4' },
    { label: 'Game 5', value: 'Game 5' },
  ]}
/>
```

Checkbox options with item count

```jsx react-live
<CheckboxMultiselect
  onChange={(value) => console.log('value: ', value)}
  value={['Banana', 'Pineapple']}
  options={[
    { label: 'Apple', value: 'Apple', count: 5 },
    { label: 'Banana', value: 'Banana', count: 2 },
    { label: 'Mango', value: 'Mango', count: 0 },
    { label: 'Orange', value: 'Orange', count: 20 },
    { label: 'Pineapple', value: 'Pineapple', count: 3 },
  ]}
/>
```

# CheckboxMultiselect

Group of checkbox options

```jsx react-live
<CheckboxMultiselect
  onChange={value => console.log(value)}
  clearSelectedLabel={<span>Clear selected items</span>}
  value={[]}
  options={[
    { label: 'Game 1' },
    { label: 'Game 2' },
    {
      label: (
        <span>
          Game <b>3</b>
        </span>
      ),
    },
    {
      label: (
        <span>
          Game <b>4</b>
        </span>
      ),
    },
    {
      label: (
        <span>
          Game <b>5</b>
        </span>
      ),
    },
  ]}
/>
```

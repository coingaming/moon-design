# Button

## Modifiers

### Primary

To be used where the the button would be the main call to action on the page.

<Playground>
  <Button modifier="primary" spaced>
    Default
  </Button>
  <Button modifier="primary" mockState="active" spaced>
    Hover
  </Button>
  <Button modifier="primary" mockState="focus" spaced>
    Focus
  </Button>
  <Button modifier="primary" disabled spaced>
    Disabled
  </Button>
</Playground>

## Sizing

### Full Width

Stretches a button to fill the entire width of its container.

<Playground>
  <Button modifier="secondary" fullWidth>
    Full Width
  </Button>
</Playground>

## Spacing

All examples in this documentation use the `spacing` prop to apply consistent right and bottom margins. Removing this will allow the button to be displayed flush with other elements.

## Elements

### Anchor

Sometimes we want to use the Button component to send the user to a different location.

By adding the `href="<url>"` attribute, Buttons can be rendered as `<a>` elements.

<Playground>
  <Button href="#" modifier="secondary">
    Secondary (Anchor)
  </Button>
</Playground>

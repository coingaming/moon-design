# Button

## Modifiers

### Primary

To be used where the the button would be the main call to action on the page.

```jsx react-live
<Inline>
  <Button modifier="primary">Default</Button>
  <Button modifier="primary" mockState="hover">
    Hover
  </Button>
  <Button modifier="primary" mockState="active">
    Focus
  </Button>
  <Button modifier="primary" disabled>
    Disabled
  </Button>
  <Loader />
</Inline>
```

## Uppercase

To be used where the the button caption would be uppercased.

```jsx react-live
<Inline>
  <Button modifier="primary" uppercase>
    Uppercased primary
  </Button>
  <Button modifier="secondary" uppercase>
    Uppercased secondary
  </Button>
</Inline>
```

## Sizing

### Full Width

Stretches a button to fill the entire width of its container.

```jsx react-live
<Button modifier="secondary" fullWidth>
  Full Width
</Button>
```

## Elements

### Anchor

Sometimes we want to use the Button component to send the user to a different location.

By adding the `as="a"` attribute, Buttons can be rendered as `<a>` elements.

Don't forget to pass `href="<url>"`

```jsx react-live
<Button href="#" as="a" modifier="secondary">
  Secondary (Anchor)
</Button>
```

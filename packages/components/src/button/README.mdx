# Button

## Modifiers

### Primary

To be used where the the button would be the main call to action on the page.

```jsx react-live
<div>
  <Button modifier="primary" spaced>
    Default
  </Button>
  <Button modifier="primary" mockState="active" spaced>
    Hover
  </Button>
  <Button modifier="primary" mockState="focus" spaced>
    Focus
  </Button>
  <Button modifier="primary" disabled spaced>
    Disabled
  </Button>
</div>
```

### Secondary

To be used where the the button would be alternative or secondary call to action.

```jsx react-live
<div>
  <Button modifier="secondary" spaced>
    Default
  </Button>
  <Button modifier="secondary" mockState="active" spaced>
    Hover
  </Button>
  <Button modifier="secondary" mockState="focus" spaced>
    Focus
  </Button>
  <Button modifier="secondary" disabled spaced>
    Disabled
  </Button>
</div>
```

### Optional

To be used where the the button would be an optional call to action - meaning not prominant.

```jsx react-live
<div>
  <Button modifier="optional" spaced>
    Default
  </Button>
  <Button modifier="optional" mockState="active" spaced>
    Hover
  </Button>
```

```jsx react-live

```

## Sizing

### Full Width

Stretches a button to fill the entire width of its container.

```jsx react-live
<Button modifier="secondary" fullWidth>
  Full Width
</Button>
```

## Spacing

All examples in this documentation use the `spacing` prop to apply consistent right and bottom margins. Removing this will allow the button to be displayed flush with other elements.

## Elements

### Anchor

Sometimes we want to use the Button component to send the user to a different location.

By adding the `href="<url>"` attribute, Buttons can be rendered as `<a>` elements.

```jsx react-live
<Button href="#" modifier="secondary">
  Secondary (Anchor)
</Button>
```

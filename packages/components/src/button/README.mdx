## Variants

### Primary

To be used where the the button would be the main call to action on the page.

```jsx react-live
<Inline>
  <Button variant="primary">Default</Button>
  <Button variant="primary" mockState="hover">
    Hover
  </Button>
  <Button variant="primary" mockState="active">
    Focus
  </Button>
  <Button variant="primary" disabled>
    Disabled
  </Button>
</Inline>
```

### Secondary

To be used where the the button would be a secondary call to action.


## Sizing

### Full Width

Stretches a button to fill the entire width of its container.

```jsx react-live
<Button variant="secondary" fullWidth>
  Full Width
</Button>
```

## Elements

### Anchor

Sometimes we want to use the Button component to send the user to a different location.

By adding the `as="a"` attribute, Buttons can be rendered as `<a>` elements.

Don't forget to pass `href="<url>"`

```jsx react-live
<Button href="#" as="a" variant="secondary">
  Secondary (Anchor)
</Button>
```

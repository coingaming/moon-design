# Toggle

A basic 'bare' button that pins itself to the top right of its parent element.

It typically contains an Icon that indicates changing a component's state.

```jsx react-live
() => {
  const { color } = useTheme();

  return (
    <Toggle>
      <IconClose color={color.bulma[100]} />
    </Toggle>
  );
};
```

## Positioning

You can disable absolute position by passing `topRight=false`.

```jsx react-live
() => {
  const { color } = useTheme();

  return (
    <Toggle topRight={false}>
      <IconClose color={color.bulma[100]} />
    </Toggle>
  );
};
```

# Text

We treat semantic and visual/cosmetic decisions as two totally [separate](https://csswizardry.com/2016/02/managing-typography-on-large-apps/) things.

Our choice of `p` and `span` is a purely semantic decision, and doesn't impact cosmetics configured via the `size` prop.

## sizes

We have predefined font sizes: 10 | 12 | 14 | 16 | 18 | 20 | 24 | 32 | 48 | 56 | 64 | 72 rems. Line heights are calculated automatically based on font size.

By default font size is 16rem. You can choose any size.

```jsx react-live
<Stack>
  <Text>Text with default font-size</Text>
  <Text size={10}>Text with font-size: 10rem</Text>
  <Text size={12}>Text with font-size: 12rem</Text>
  <Text size={14}>Text with font-size: 14rem</Text>
  <Text size={16}>Text with font-size: 16rem</Text>
  <Text size={18}>Text with font-size: 18rem</Text>
  <Text size={20}>Text with font-size: 20rem</Text>
  <Text size={24}>Text with font-size: 24rem</Text>
  <Text size={32}>Text with font-size: 32rem</Text>
  <Text size={48}>font-size: 48rem</Text>
  <Text size={56}>font-size: 56rem</Text>
  <Text size={64}>font-size: 64rem</Text>
  <Text size={72}>font-size: 72rem</Text>
</Stack>
```

## isBold

By default font weight is regular. You can make it bolder.

```jsx react-live
<Stack>
  <Text size={24}>Text with default font weight</Text>
  <Text size={24} isBold>
    Text with font-weight: semibold
  </Text>
</Stack>
```

## as="span"

By default Text renders as Paragraph tag. You can render it as Span or Label tags, etc.

```jsx react-live
<Stack>
  <Text size={24}>Text renders Paragraph tag by default</Text>
  <Text size={24} as="span">
    Text renders Span tag
  </Text>
  <Text size={24} as="label">
    Text renders Label tag
  </Text>
</Stack>
```

## color

By default Text color is bulma.100. You can use any color.

Bulma.100 - is main text color. Trunks.100 - is secondary text color

```jsx react-live
<Stack>
  <Text size={24}>Text with default color</Text>
  <Text size={24} color="trunks.100">
    Text with defined color
  </Text>
  <Text size={24} color="piccolo.100">
    Text with defined color
  </Text>
  <Text size={24} color="krillin.100">
    Text with defined color
  </Text>
</Stack>
```

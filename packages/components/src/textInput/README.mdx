# Text Input

The input component is used when you need to let users enter the text of some kind, such as their name or phone number etc.

```jsx react-live
<TextInput label="Text Input" placeholder="e.g. username" />
```

All text inputs must have visible labels/placeholder text.

[Figma Link](https://www.figma.com/file/S3q1SkVngbwHuwpxHKCsgtJj/Global?node-id=5400%3A1634)

## Types

For mobile, the usage of respected keyboards is recommended,
f.e. if it’s likely that the user will need to enter a number and you want to bring up
the numeric keypad on a mobile device.

Text inputs support a range of input formats, some of which include automatic
browser validation. These can be set by providing HTML input `type` attributtes:

```jsx react-live
<Stack style={{ maxWidth: '25rem' }}>
  <TextInput type="text" label="Text" placeholder="e.g. username" />

  <TextInput type="date" label="Date" />

  <TextInput type="datetime-local" label="Date and time" />

  <TextInput
    type="email"
    label="Email"
    placeholder="e.g. jake.weary@sportsbet.io"
  />

  <TextInput type="number" label="Number" placeholder="e.g. 1234" required />

  <TextInput
    type="password"
    placeholder="Please enter your account password"
    label="Password"
  />

  <TextInput type="search" label="Search" placeholder="e.g. EPL" required />

  <TextInput type="tel" label="Telephone" placeholder="e.g. +372 123 4567" />

  <TextInput type="time" label="Time" />

  <TextInput type="url" label="URL" placeholder="e.g. https://sportsbet.io" />
</Stack>
```

## Validation

Text inputs offer different states to provide instant feedback to the user.

Errors are automatically handled as part of the HTML input type validation, and
can also be manually triggered via the `error` prop.

```jsx react-live
<Stack style={{ maxWidth: '25rem' }}>
  <TextInput
    type="email"
    label="Email (with default validation)"
    placeholder="e.g. jake.weary@sportsbet.io"
    required
  />

  <TextInput
    type="email"
    label="Email (with error prop)"
    placeholder="e.g. jake.weary@sportsbet.io"
    error
  />

  <TextInput
    type="email"
    label="Email (with error prop)"
    placeholder="e.g. jake.weary@sportsbet.io"
    error="User already exist"
  />
</Stack>
```

### Icon

SVG Icon can also be manually triggered via the `icon` prop.
You could check icons [here](/assets/icons 'here').

```jsx react-live
<Stack style={{ maxWidth: '25rem' }}>
  <TextInput
    type="text"
    label="Text"
    placeholder="e.g. username"
    icon={<IconGeneralInfo fontSize="10rem" color="#8697A2" />}
  />
</Stack>
```

### Disabled

Passing a standard HTML `disabled` attribute will render an input disabled,
preventing user interaction.

```jsx react-live
<Form maxWidth="25rem">
  <FormItem>
    <TextInput
      type="text"
      label="Disabled Input"
      placeholder="e.g. username"
      disabled
    />
  </FormItem>
</Form>
```

### Right direction

```jsx react-live
<Form maxWidth="25rem">
  <FormItem>
    <TextInput
      dir="rtl"
      type="text"
      label="آراء وتحليلات."
      placeholder="e.g. username"
    />
  </FormItem>
</Form>
```

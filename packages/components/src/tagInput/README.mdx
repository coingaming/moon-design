# TagInput

Input field with tags and suggestions
[Figma link](https://www.figma.com/file/rQDkupIZuIlueXNMoYObjt/Hub88-Templates?node-id=1942%3A12077)

```jsx react-live
<TagInputWrapper>
  <TagInputMain>
    <TagInputField placeholder="Hint text" />
    <IconDropdown />
  </TagInputMain>
</TagInputWrapper>
```

### with visible dropdown

```jsx react-live
<TagInputWrapper>
  <TagInputMain focused={true}>
    <TagInputField placeholder="Hint text" />
    <IconDropdown />
  </TagInputMain>

  <DropdownWrapper style={{ position: 'relative' }}>
    <FilterDropdown>
      <CheckboxMultiselect
        onChange={value => console.log(value)}
        value={[]}
        options={[
          { label: 'Label 1', value: 1 },
          { label: 'Label 2', value: 2 },
          { label: 'Label 3', value: 3 },
        ]}
      />
    </FilterDropdown>
  </DropdownWrapper>
</TagInputWrapper>
```

### with selected tags

```jsx react-live
<TagInputWrapper>
  <TagInputMain focused={true}>
    <Tag>
      Label 1
      <TagRemove>
        <IconCloseRounded />
      </TagRemove>
    </Tag>
    <Tag>
      Label 2
      <TagRemove>
        <IconCloseRounded />
      </TagRemove>
    </Tag>
    <TagInputField />
    <RemoveAllButton>
      <IconCloseRounded />
    </RemoveAllButton>
  </TagInputMain>

  <DropdownWrapper style={{ position: 'relative' }}>
    <FilterDropdown>
      <CheckboxMultiselect
        onChange={value => console.log(value)}
        value={[{ label: 'Label 1', value: 1 }, { label: 'Label 2', value: 2 }]}
        options={[
          { label: 'Label 1', value: 1 },
          { label: 'Label 2', value: 2 },
          { label: 'Label 3', value: 3 },
        ]}
      />
    </FilterDropdown>
  </DropdownWrapper>
</TagInputWrapper>
```

### with hidden dropdown

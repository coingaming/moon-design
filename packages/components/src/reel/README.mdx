# Reel

Allows content to overflow on the **horizontal axis**, forming the base for more
complex UI components such as [`Carousel`](./carousel).

See [every-layout.dev](https://every-layout.dev/layouts/reel/) for further
documentation of the "reel".

```jsx react-live
() => {
  const Example = styled.div(({ theme: { color, radius }, hueRotate }) => ({
    borderRadius: radius.small,
    width: '20rem',
    height: '12rem',
    backgroundColor: color.piccolo[100],
    filter: `hue-rotate(${hueRotate}deg)`,
  }));

  const items = Array.from({ length: 50 }, (v, i) => i);

  return (
    <Reel>
      {items.map(item => (
        <Example hueRotate={item} />
      ))}
    </Reel>
  );
};
```

## Customization

### Space

Spacing between adjacent content can be adjusted via the `space` prop.

`space` accepts themed space keys (e.g. `large` returns the large themed `space`
key in `rem`), a number (returned in `px`) or a string value (e.g. `"30px"` or
`"1rem"`).

```jsx react-live
() => {
  const Example = styled.div(({ theme: { color, radius }, hueRotate }) => ({
    borderRadius: radius.small,
    width: '20rem',
    height: '12rem',
    backgroundColor: color.piccolo[100],
    filter: `hue-rotate(${hueRotate}deg)`,
  }));

  const items = Array.from({ length: 50 }, (v, i) => i);

  return (
    <Reel space="large">
      {items.map(item => (
        <Example hueRotate={item} />
      ))}
    </Reel>
  );
};
```

### Height

```jsx react-live
() => {
  const placeholderImg = size => `https://www.placecage.com/${size}`;

  return (
    <Reel height="20vh">
      <img alt="" src={placeholderImg('200/400')} />
      <img alt="" src={placeholderImg('400/300')} />
      <img alt="" src={placeholderImg('200/200')} />
      <img alt="" src={placeholderImg('150/200')} />
      <img alt="" src={placeholderImg('450/300')} />
      <img alt="" src={placeholderImg('200/200')} />
    </Reel>
  );
};
```

### Navigation

Using `Reel`, we can quickly mock-up a set of ["sausage links"](https://uglyduck.ca/hamburger-menu-alternative/) for navigation.

To ensure semantics, render the component `as="ul"`. All child
list-item (`li`) elements will display without bullets.

```jsx react-live
() => {
  const items = Array.from({ length: 20 }, (v, i) => i);

  return (
    <Reel as="ul">
      {items.map(item => (
        <li>
          <Button variant="primary">Item {item}</Button>
        </li>
      ))}
    </Reel>
  );
};
```

# Link

Links are used to navigate users through the user-interface via anchors or embedded actions.

```jsx react-live
<Link href="#">I'm a link</Link>
```

## Secondary

```jsx react-live
<Link href="#" secondary>
  I'm a Secondary link
</Link>
```

## Optional

```jsx react-live
<Link href="#" optional>
  I'm an Optional link
</Link>
```

## Elements

### Button

Sometimes it might make more semantic sense to render the Link as a `<button>` element.

This can be achieved by **adding** the `as="button"` attribute.

```jsx react-live
<Link as="button">I look like a link, but really I'm a button</Link>
```

## DisabledTW

Links that cannot be selected can be given a disabled state.

```jsx react-live
<Link href="#" disabled>
  I'm a disabled link
</Link>
```

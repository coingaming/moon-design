# TagList

List of tags with expanding dropdown
[Figma link](https://www.figma.com/file/rQDkupIZuIlueXNMoYObjt/Hub88---Templates?node-id=2017%3A0)

```jsx react-live
<TagListWrapper>
  <List>
    <Tag>Bender</Tag>
    <Tag>Livegames</Tag>
    <Tag>Bitcasino</Tag>
  </List>
</TagListWrapper>
```

## with "show more" button

```jsx react-live
<TagListWrapper>
  <List>
    <Tag>Bender</Tag>
    <Tag>Livegames</Tag>
    <Tag>Bitcasino</Tag>
    <ShowMore>+6</ShowMore>
  </List>
</TagListWrapper>
```

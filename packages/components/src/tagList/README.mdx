# TagList

List of tags with expanding dropdown
[Figma link](https://www.figma.com/file/rQDkupIZuIlueXNMoYObjt/Hub88---Templates?node-id=2017%3A0)

```jsx react-live
<TagListWrapper>
  <TagListInternal>
    <Tag>Bender</Tag>
    <Tag>Livegames</Tag>
    <Tag>Bitcasino</Tag>
  </TagListInternal>
</TagListWrapper>
```

## with "show more" button

```jsx react-live
<TagListWrapper>
  <TagListInternal>
    <Tag>Bender</Tag>
    <Tag>Livegames</Tag>
    <Tag>Bitcasino</Tag>
    <ShowMore>+6</ShowMore>
  </TagListInternal>
</TagListWrapper>
```

## with expanded list

```jsx react-live
<Inline style={{ padding: '100px 16px' }}>
  <TagListWrapper>
    <TagListInternal>
      <Tag>Bender</Tag>
      <Tag>Livegames</Tag>
      <Tag>Bitcasino</Tag>
      <ShowMore>+6</ShowMore>
    </TagListInternal>
    <Dropdown style={{ width: '120%' }}>
      {Array.from({ length: 30 }).map((_, i) => (
        <Tag key={`${i}`}>Item {i}</Tag>
      ))}
      <CopyWrapper>
        <CopyButton>
          <CopyBubble position="bottom">Copy</CopyBubble>
          <IconCopy color="trunks.100" />
        </CopyButton>
      </CopyWrapper>
    </Dropdown>
  </TagListWrapper>
</Inline>
```

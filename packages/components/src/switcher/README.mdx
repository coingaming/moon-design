# Switcher

An alternate checkbox appearance for simulating on/off state.

```jsx react-live
<Switcher />
```

An additional variant of the `Switch` can be used to toggle the app's
`colorScheme` between 'light' and 'dark'.

```jsx react-live
<Switcher colorScheme />
```

## Captions

For extra context, captions can be appended to the unchecked and checked
positions of the switcher.

```jsx react-live
<Switcher captionUnchecked="ON" captionChecked="OFF" />
```

```jsx react-live
<Switcher colorScheme captionUnchecked="AM" captionChecked="PM" />
```

## State

Any manipulation of state should be handled via `onChange`. Below is an example
with React Hooks:

```jsx react-live
() => {
  const [state, setState] = React.useState(true);

  return <Switcher checked={state} onChange={e => setState(e.target.checked)} />;
};
```

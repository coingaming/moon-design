# Form

Handling user-submitted information.

## Anatomy

1. **Form**

   The main form element renders items as a list inside a `<fieldset>`, along with a required hidden `legend` for accessibility. It allows customisation of `maxWidth` and accepts all standard HTML `<form>` attributes.

2. **Form Item**

   A wrapper for each top-level input or fieldset.

3. **Form Error**

   A generic text component for diplaying error messages to the user.

```jsx
<Form legend="Example form" action="#">
  <FormItem>{/* input|fieldset */}</FormItem>
  <FormItem>
    {/* input|fieldset */}
    <FormError>Invalid password!</FormError>
  </FormItem>
</Form>
```

### Example

An example sign-up form, making use of other input components to collect user-entered information.

```jsx react-live
<Form legend="Sign-up" maxWidth="25rem">
  <FormItem>
    <TextInput label="Name" type="text" placeholder="e.g. John Appleseed" />
  </FormItem>
  <FormItem>
    <TextInput
      error
      label="Email"
      type="email"
      placeholder="e.g. john@coingaming.io"
    />
    <FormError>Please enter a valid email address.</FormError>
  </FormItem>
  <FormItem>
    <Select label="Country of residence" flex>
      <option value="">Select a country</option>
      <option value="estonia">Estonia</option>
      <option value="united-kingdom">United Kingdom</option>
      <option value="other">Other</option>
    </Select>
  </FormItem>
  <FormItem>
    <Checkbox label="I agree to the terms and conditions" />
  </FormItem>
  <FormItem>
    <Button variant="primary" fullWidth>
      Sign up
    </Button>
  </FormItem>
</Form>
```

## Combo

For micro-journeys, consider using a combo item.

```jsx react-live
<Form legend="Enable Two-Factor Authentication" maxWidth="25rem">
  <FormItem>
    <FormCombo>
      <FormComboInput>
        <TextInput placeholder="Enter code" />
      </FormComboInput>
      <FormComboButton>
        <Button>Submit</Button>
      </FormComboButton>
    </FormCombo>
  </FormItem>
</Form>
```

When including an error message, append below the `FormCombo` element

```jsx react-live
<Form legend="Enable Two-Factor Authentication" maxWidth="25rem">
  <FormItem>
    <FormCombo>
      <FormComboInput>
        <TextInput error placeholder="Enter code" />
      </FormComboInput>
      <FormComboButton>
        <Button>Submit</Button>
      </FormComboButton>
    </FormCombo>
    <FormError>You have entered an invalid code.</FormError>
  </FormItem>
</Form>
```

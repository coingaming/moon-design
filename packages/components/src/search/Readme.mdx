# Search

An elegant search component.

## Query

Use `query` prop to define query string

`onChange` and `onSubmit` handlers are provided.

<Search
  placeholder="Search"
  onChange={(e) => {
    console.log(e.target.value);
  }}
  onSubmit={(e) => {
    console.log('Submitted');
  }}
/>

### Results

Pass results to `results` prop.

```jsx react-live
<Search
  placeholder="Search"
  onChange={(e) => {
    console.log(e.target.value);
  }}
  onSubmit={(e) => {
    console.log('Submitted');
  }}
  results={[
    {
      title: <span>Games</span>,
      items: [
        <span
          onClick={() => {
            console.log('Click');
          }}
        >
          First Game
        </span>,
        <span>Second Game</span>,
        <span>Third Game</span>,
      ],
    },
    {
      title: <span>Providers</span>,
      items: [
        <span>Provider One</span>,
        <span>Provider Two</span>,
        <span>Provider Three</span>,
      ],
    },
  ]}
/>
```

### Loading Message

Pass `loadingMessage`.
loadingMessage has a priority and even if you define results
they won't be rendered.

```jsx react-live
<Search
  placeholder="Search"
  onChange={(e) => {
    console.log(e.target.value);
  }}
  loadingMessage={<p>Loading...</p>}
/>
```

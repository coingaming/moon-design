# Select

A user-controlled menu of options for forms, navigation and more.

```jsx react-live
<Select>
  <option value="">Select an option</option>
  <option value="never-gonna">Never gonna</option>
  <option value="give-you-up">give you up</option>
  <option value="let-you-down">let you down</option>
</Select>
```

## Size

### Full Width

By default, Select stretches to full width on small devices. For Select inputs that need to remain full width on all devices, use the `fullWidth` prop.

```jsx react-live
<Select fullWidth>
  <option value="">Select an option</option>
  <option value="never-gonna">Never gonna</option>
  <option value="give-you-up">give you up</option>
  <option value="let-you-down">let you down</option>
</Select>
```

### Short

For shorter Select options, such as currency units, make use of the `unit` variant.

```jsx react-live
<Select short>
  <option value="eur">EUR</option>
  <option value="gbp">GBP</option>
  <option value="usd">USD</option>
</Select>
```

## States

### Disabled

Select inputs that cannot be interacted with should be given a `disabled` state.

```jsx react-live
<Select disabled>
  <option value="">Select an option</option>
  <option value="never-gonna">Never gonna</option>
  <option value="give-you-up">give you up</option>
  <option value="let-you-down">let you down</option>
</Select>
```

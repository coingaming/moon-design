# Card

Containers for mixed content and media about a single subject.

## Templates

### Front

Default template, using the `flex` layout modifier to balance space between items.

<Playground>
  <Card flex>
    <CardToggle />
    <CardBalance
      from={{
        currency: 'Bitcoin',
        value: 1526.56,
        unit: 'mBTC',
        unitTitle: 'Millibitcoin',
      }}
      to={{ currency: 'Dollars', value: 354568.68, unit: 'USD' }}
    />
    <button>Deposit</button>
    <button>Withdraw</button>
  </Card>
</Playground>

### Back

The other 'flipped' side of the card.

<Playground>
  <Card template="back">
    <CardToggle back />
    <form>
      <ul>
        <li>
          <label for="currency-converted">Converted currency</label>
          <select id="currency-converted" name="currency-converted">
            <option value="euro">Euro</option>
            <option value="gbp">GBP</option>
            <option value="usd">USD</option>
          </select>
        </li>
        <li>
          <label for="currency-balance">Balance in</label>
          <select id="currency-balance" name="currency-balance">
            <option value="euro">mBTC</option>
            <option value="gbp">Doge</option>
            <option value="usd">Ethereum</option>
          </select>
        </li>
      </ul>
    </form>
  </Card>
</Playground>

### Outline

An 'empty' ghost outline of the card.

<Playground>
  <Card template="outline">
    <button>+ Add Wallet</button>
  </Card>
</Playground>

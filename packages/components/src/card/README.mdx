# Card

Containers for mixed content and media about a single subject.

## Templates

### Front

Default template, using the `flex` layout modifier to balance space between items.

[Icons](../icons) can also be passed via `backgroundIcon` for extra cosmetic context.

```jsx react-live
<Card flex backgroundIcon={<IconBitcoin />}>
  <CardToggle />
  <CardBalance
    from={{
      currency: 'Bitcoin',
      value: '1526.56',
      unit: 'mBTC',
      unitTitle: 'Millibitcoin',
    }}
  />
  <div>
    <Link style={{ marginRight: spacing() }}>Deposit</Link>
    <Link disabled>Withdraw</Link>
  </div>
</Card>
```

### Back

The other 'flipped' side of the card.

```jsx react-live
<Card template="back">
  <CardToggle back />
  <Form legend="Convert currency" maxWidth="18rem">
    <FormItem>
      <Select label="Converted currency" flex>
        <option value="euro">Euro</option>
        <option value="gbp">GBP</option>
        <option value="usd">USD</option>
      </Select>
    </FormItem>
    <FormItem>
      <Select label="Balance in" flex>
        <option value="euro">mBTC</option>
        <option value="gbp">Doge</option>
        <option value="usd">Ethereum</option>
      </Select>
    </FormItem>
  </Form>
</Card>
```

### Outline

An 'empty' ghost outline of the card.

```jsx react-live
<Card template="outline">
  <button>+ Add Wallet</button>
</Card>
```

### Badge

Set a `badgeCaption` prop

```jsx react-live
<Card flex backgroundIcon={<IconBitcoin />}>
  <CardToggle />
  <CardBalance
    badgeCaption="Active"
    from={{
      currency: 'Bitcoin',
      value: '1526.56',
      unit: 'mBTC',
      unitTitle: 'Millibitcoin',
    }}
  />
  <div>
    <Link style={{ marginRight: spacing() }}>Deposit</Link>
    <Link disabled>Withdraw</Link>
  </div>
</Card>
```

## Anatomy

### Balance

Displays the original and converted wallet balance.

```jsx react-live
<CardBalance
  from={{
    currency: 'Bitcoin',
    value: '1526.56',
    unit: 'mBTC',
    unitTitle: 'Millibitcoin',
  }}
/>
```

<PropsTable of={CardBalance} />

### Toggle

An extension of `Toggle` that can be used to offer users with the option to toggle between 'settings' and to go `back` to the front of the Card.

```jsx react-live
<CardToggle />
```

# Card

Containers for mixed content and media about a single subject.

## Templates

### Front

Default template, using the `flex` layout modifier to balance space between items.

[Icons](../icons) can also be passed via `backgroundIcon` for extra cosmetic context.

```jsx react-live
<Card flex backgroundIcon={<IconBitcoin />}>
  <CardBalance
    from={{
      currency: 'Bitcoin',
      value: '1526.56',
      unit: 'mBTC',
      unitTitle: 'Millibitcoin',
    }}
  />
  <div>
    <Link optional style={{ marginRight: spacing() }}>
      Deposit
    </Link>
    <Link optional disabled>
      Withdraw
    </Link>
  </div>
</Card>
```

### Outline

An 'empty' ghost outline of the card.

```jsx react-live
<Card template="outline">
  <button>+ Add Wallet</button>
</Card>
```

## Anatomy

### Balance

Displays the original and converted wallet balance.

```jsx react-live
<CardBalance
  from={{
    currency: 'Bitcoin',
    value: '1526.56',
    unit: 'mBTC',
    unitTitle: 'Millibitcoin',
  }}
/>
```

#### Badge

An additional badge can be passed via the `badgeCaption` prop.

```jsx react-live
<CardBalance
  badge={<Badge>Active</Badge>}
  from={{
    currency: 'Bitcoin',
    value: '1526.56',
    unit: 'mBTC',
    unitTitle: 'Millibitcoin',
  }}
/>
```

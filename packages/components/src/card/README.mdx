# Card

### ⚠️**Deprecated**

Containers for mixed content and media about a single subject.

## Templates

### Front

Default template, using the `flex` layout modifier to balance space between items.

[Icons](../icons) can also be passed via `backgroundIcon` for extra cosmetic context.

```jsx react-live
() => {
  const Example = styled.div({
    maxWidth: 300,
  });

  return (
    <Example>
      <Card flex backgroundIcon={<IconCurrencyTrx />}>
        <CardBalance
          from={{
            currency: 'Bitcoin',
            value: '1526.56',
            unit: 'mBTC',
            unitTitle: 'Millibitcoin',
          }}
        />
        <Inline>
          <Link optional>Deposit</Link>
          <Link optional disabled>
            Withdraw
          </Link>
        </Inline>
      </Card>
    </Example>
  );
};
```

### Outline

An 'empty' ghost outline of the card.

```jsx react-live
() => {
  const Example = styled.div({
    maxWidth: 300,
  });

  return (
    <Example>
      <Card template="outline">
        <button>+ Add Wallet</button>
      </Card>
    </Example>
  );
};
```

## Anatomy

### Balance

Displays the original and converted wallet balance.

```jsx react-live
<CardBalance
  from={{
    currency: 'Bitcoin',
    value: '1526.56',
    unit: 'mBTC',
    unitTitle: 'Millibitcoin',
  }}
/>
```

#### Badge

An additional badge can be passed in via the `badge` prop.

```jsx react-live
<CardBalance
  badge={<Badge>Active</Badge>}
  from={{
    currency: 'Bitcoin',
    value: '1526.56',
    unit: 'mBTC',
    unitTitle: 'Millibitcoin',
  }}
/>
```

# FilterDropdown

Wrapper of different filters

```jsx react-live
<FilterDropdown
  searchPlaceholder="Search for a name..."
  onSearchChange={value => console.log(value)}
></FilterDropdown>
```

## Without Search bar

```jsx react-live
<FilterDropdown>
  <Text>
    Lorem ipsum dolor sit amet, consectetur <br />
    adipiscing elit. Praesent volutpat.
  </Text>
</FilterDropdown>
```

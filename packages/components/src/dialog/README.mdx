# Dialog

A small pop-up window to display additional content or request action from the
user.

For consistency, make use of the additional `DialogHeading` and `DialogText`
components for content; using these inside a [`Stack`](/components/stack)
will offer the bonus of consistent vertical rhythm.

```jsx react-live
() => {
  const [showDialog, setShowDialog] = React.useState(false);

  return (
    <>
      <Button modifier="secondary" onClick={() => setShowDialog(true)}>
        Open Dialog
      </Button>
      <Dialog isOpen={showDialog} onDismiss={() => setShowDialog(false)}>
        <Stack>
          <DialogHeading>Lorem ipsum dolor sit amet.</DialogHeading>
          <DialogText>
            Consectetur adipiscing elit. Phasellus eget vulputate nibh, nec
            laoreet lectus. Proin at lacus quis justo mattis porta. Nulla sed
            nisl volutpat, varius enim eu, semper augue.
          </DialogText>
        </Stack>
      </Dialog>
    </>
  );
};
```

## State

Dialog is a styled extension of Reach UI's "Dialog" component, with state
handled via `isOpen` and `onDismiss`. More information on these props, can be
found in the [Reach UI documentation](https://ui.reach.tech/dialog).

Examples on this page show usage with
[React Hooks](https://reactjs.org/docs/hooks-intro.html), but this could also be
handled via higher-order components or libaries such as
[Recompose](https://github.com/acdlite/recompose).

## Footer

For cases where a dialog requires additional action from the user, elements such
as buttons can be passed into the `footer` prop.

```jsx react-live
() => {
  const [showDialog, setShowDialog] = React.useState(false);
  const open = () => setShowDialog(true);
  const close = () => setShowDialog(false);

  return (
    <>
      <Button modifier="secondary" onClick={open}>
        Open Dialog
      </Button>
      <Dialog
        isOpen={showDialog}
        onDismiss={close}
        footer={
          <Stack direction="horizontal" space={spacing('small')}>
            <Button modifier="primary" onClick={close}>
              Save
            </Button>
            <Button modifier="secondary" onClick={close}>
              Cancel
            </Button>
          </Stack>
        }
      >
        <Stack>
          <DialogHeading>Lorem ipsum dolor sit amet.</DialogHeading>
          <DialogText>
            Consectetur adipiscing elit. Phasellus eget vulputate nibh, nec
            laoreet lectus. Proin at lacus quis justo mattis porta. Nulla sed
            nisl volutpat, varius enim eu, semper augue.
          </DialogText>
        </Stack>
      </Dialog>
    </>
  );
};
```

## Customization

To adjust the dialog's default appearance for individual use-cases, make use of:

- `maxWidth`: adjusts the overall maximum width of the dialog window.

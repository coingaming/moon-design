# Introduction

This package houses all themes you might need to build an application based on Tailwindcss.

## Installing

```bash
pnpm install --save @heathmont/moon-themes-tw
```

## Consuming

```javascript
@import '@heathmont/moon-themes-tw/lib/moon.css';
@import '@heathmont/moon-themes-tw/lib/sportsbet.css';
@import '@heathmont/moon-themes-tw/lib/bitcasino.css';
@import '@heathmont/moon-themes-tw/lib/betadda.css';
```

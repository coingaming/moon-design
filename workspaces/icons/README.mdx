# Introduction

This package houses all **SVG icons** you might need to build an application based on Tailwindcss (without styled-component dependency).

## Installing

```bash
pnpm install --save @heathmont/moon-icons-tw
```

## Consuming

```javascript
import {
  SportBeachVolley,
  ChartDashboard,
  ArrowsLeftCurved,
} from '@heathmont/moon-icons-tw';
```
